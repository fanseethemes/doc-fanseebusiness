INTRODUCTION

Bizart Pro is a light multipurpose business WordPress theme that comes with a smooth design and extremely impressive functionality to unleash your website. It has been made to showcase any services that you provide. You can use it for any kind of website and deliver an awesome look. Bizart Pro includes both the looks and astounding functionality to further intensify the website.

The new multipurpose business theme, Bizart Pro is packed with features like Featured Content, Hero Content, What We Do, Testimonials, Portfolios, FAQs and more. All of these features have been set to keep multipurpose business websites in mind. Bizart Pro is the ultimate stop if you want a light and bold theme for your website.

Bizart pro provides easy way to costomize the theme. Please follow the below steps to navigate homepage settings.

ADD NEW SLIDER

Check Video tutorial
  • Go to the Dashboard and click on sliders menu and click on the Add New
  • Now, Add title for your banner on "Add title". For sub title, type text on "Sub Text" as shown in the figure below.
  • To set Banner image, click on Set Featured Image You can select select the image from Media Library or can upload images from Upload files.
  • After selecting the suitable image, click on Set Featured Image as shown in the figure below.
  • Two banner buttons are included in each slider which are by default disabled. To enable banner button on slider, tick on "Enable Button (First)". Likewise to enable both banner button, tick on "Enable Button (Second)". Both banner button includes Button Text and Button Link as shown in the figure below.

INTEGRATION

Integration feature give you to customize the slider as your requirements. Customize your slider as your requirement and add on you custom page (By adding provided shortcode) or update the customizer option from this page too. which is super easy.

  • To customize slider, Go to Dashboard -> Sliders -> Integration.
  • Layout: You can select different layout as per your choice. Here, we have only default layout for now as we are working on other layouts.
  • Limit: This feature helps to limit how many sliders you want to show in your site. You can change the limitation as per your requirement.
  • Enable Permalink: It helps to enable or disable the permalink on the sliders.
  • Show Dots: It helps to enable or disable the dots shown on the sliders.
  • Show Arrows: It helps to enable or disable the arrow on the sliders. This arrow helps to show another slider consecutively.
  • Infinite: It helps to create infinite loop for the sliders.
  • Autoplay: It helps to autoplay the sliders when enabled.
  • Curve Design: It helps to enable or disable the curve design below the slider.
  • Fade: It helps to enable or disable the fade effect on the slider.
  • Button Background Color: It helps to select the colors of buttons on the sliders.
  • Button Background Hover Color: It helps to select the hover color of buttons on the sliders.
  • Overlay Color: It helps to select the overlay color of the slider.
  • Shortcode Generator: It helps to generate the code which can be used in any pages.
  • Save to Customizer: After the changes are done, this button helps to save the changes made.

ADD NEW SERVICE

  • Go to the Dashboard and click on Services menu and click on the Add New
  • Now, add title for your service on "Add title".
  • To set the service image, select an icon image fromSelect Image
  • Now, select the image icon and click onInsert
  • At last, Click on Publish

INTEGRATION

Integration feature helps you to customize the services as per your requirements. This feature also helps to add on your custom page (By adding provided shortcode) or update the customizer option from this page too. which is super easy.

  • To customize slider, Go to Dashboard -> Services -> Integration.
  • General Settings includes following features.
    • Title: This option helps to write the title of the service.
    • Sub Title: This option helps to write the subtitle of the service.
    • Description: This option includes the description about the service you want to include.
    • Button Text: This button helps to show more detail information about the service you want to include.
    • Save Changes: After the changes are done in title, subtitle, description and button text, this button helps to save the changes made.
  • Shortcode Generator includes following features.
    • Limit: This feature helps to limit how many services you want to show. You can change the limitation as per your requirement.
    • Layout: You can select different layout for services. But we have only default layout for now as we are working on other layouts.
    • Code Generator: It helps to generate the code which can be used in any custom pages.
    • Save to Customizer: After the changes are done in limit, layout and code generator, this button helps to save the changes made.
