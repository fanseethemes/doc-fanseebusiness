INTRODUCTION

Fansee Business is a light multipurpose business WordPress theme that comes with a smooth design and extremely impressive functionality to unleash your website. It has been made to showcase any services that you provide. You can use it for any kind of website and deliver an awesome look. Fansee Business includes both the looks and astounding functionality to further intensify the website.

The new multipurpose business theme, Fansee Business is packed with features like Featured Content, Hero Content, What We Do, Testimonials, Portfolios, FAQs and more. All of these features have been set to keep multipurpose business websites in mind. F is the ultimate stop if you want a light and bold theme for your website.

Home page customizer

  • To customize the home page, first go to Dashboard then click on Appearance and click Customize.
    After clicking on "Customize", options like front page options, theme options, site identity, background image menus etc will be listed as shown in the figure below.

Front Page Option

  • Front page option helps to customize the look of front page. You can customize front page by customizing following options shown in figure.

General

  • This option helps to customize the general settings of the front page. It includes SVG shape color and show front page content.
  • General settings includes following features.
    • SVG Shape Color: It helps to change the color of the page.
    • Show Front Page Content: It helps to enable or disable the front page content.
      When we enable the "Show Front Page Content", another "Show Content Above" option gets enabled automatically. This option helps to enable or disable the content above the page.

Slider

  • This option helps to customize the slider settings of the front page.
  • Slider settings includes following features.
    • Enable Slider: It helps to show the slider on the front page.
    • Pages: It helps to add or delete the page shown in the slider of the front page as per requirement.
    • Autoplay: It helps to play the slider of the page automatically one after another.
    • Dots: It is shown below the slider in dot shape based on the number of pages included in the slider.
    • Infinite Scroll: It helps to keep the slider on loop when enabled.
    • Enable Shortcode: It helps to generate shortcode when enabled.

About

  • This option helps to customize the about us settings of the front page.
  • After clicking on About, following features shows up.
    • Enable: It helps to enable or disable the "About us" section on the front page.
    • Page: It helps to select the page from the pages we have listed for About us section.
    • Button Text: It helps to write text on the button available in About us section.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Service

  • This option helps to customize the Service section settings of the front page.
  • Service section settings includes following features.
    • Enable: It helps to enable or disable slider section on the front page.
    • Enable Shape: It helps to enable or disable the background shape behind the sub pages in our service section.
    • Content Page: It helps to choose the page for "Our Services" from the pages.
    • Button Text: It helps to write the text on the button available in "Our Services".
    • Sub Pages: It helps to create or delete sub pages on "Our Services" section as per requirement.
    • Enable Shortcode: It helps to generate shortcode when enabled.
